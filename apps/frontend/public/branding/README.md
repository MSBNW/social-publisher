# Custom Branding Assets

Place your brand assets here:
- Logo: social-publisher-logo.svg (or any format)
- Favicon: social-publisher-favicon.ico

Then set env vars:
- NEXT_PUBLIC_APP_LOGO_URL=/branding/your-logo.svg
- NEXT_PUBLIC_APP_FAVICON_URL=/branding/your-favicon.ico

Default (no files needed): uses stock Postiz branding.
